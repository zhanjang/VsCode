int *p      定义指针,可以指向数组
int **p     二级指针,指向一级指针
int *p[]    指针数组,一个数组都用来存指针
int (*p)[]  二维数组指针,必须加(),否则会被[]的优先级影响
int *p()    表示一个函数,它的返回值为指针变量
int (*p)()  函数指针,用一个指针变量来接收函数的地址
指针需要配合sizeof和&使用
数组指针比较大小时避免指针与数组元素最前面的指针的前一个进行比较,允许与数组元素最后的元素的后面一个进行比较
const 修饰的指针变量,修饰的是指针的本身,而非指针变量,及修饰后这个指针不在具有写入数据的权限,而非这个指针变量不可以修改
