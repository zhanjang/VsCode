10.c语言中存在数据类型的隐形转换(影响很大)
	a.单数据类型长度小于int类型时,自动转换为int计算(主要影响负数,对正数影响不大) 例如char a = 5 char b = 126 char c = a+b c结果为-125
	原因是a提升后存储为0000000000000000000000000000101 b为0000000000000000000000000111 1110
	相加为10000011 补为00000000000000000000000010000011存储截取为10000011(补码)取源码为1111 1101十进制结果为-125
	b.单长度大于int后按优先级进行转换优先级为long double > double > float > unsigned long int > long int > unsigned int > int
	与高优先级的数据类型计算时自动转换为高优先级 例如float + int 则int转换为float计算(可以通过强制转换进行干预)
